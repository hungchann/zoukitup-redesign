# Font Zelda

Để sử dụng font Zelda trong project:

1. Tải font Zelda từ: https://www.fontmirror.com/zelda
2. Giải nén file ZIP
3. Copy file `Zelda.otf` hoặc `Zelda.ttf` vào folder này (`public/fonts/`)
4. Font sẽ tự động được load qua `@font-face` trong `index.html`

Font đã được cấu hình trong Tailwind config với tên `font-zelda`.

**Lưu ý**: Font Zelda là font preview miễn phí cho mục đích cá nhân. Để sử dụng thương mại, vui lòng mua bản đầy đủ tại: https://gumroad.com/l/mQtzd
